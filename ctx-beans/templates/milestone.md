# Milestone Template

A milestone is a target release or checkpoint. It groups work that should ship together.

## Sections

### Target Date
When this milestone should be reached.

### Goals
What this milestone represents. What state will the project be in when this is hit?

### Key Deliverables
What ships or completes at this point. The concrete outputs that define this milestone.

### Exit Criteria
Conditions that must be met to consider this milestone complete.

### Risks
What could prevent reaching the milestone on time? Known blockers or uncertainties.
