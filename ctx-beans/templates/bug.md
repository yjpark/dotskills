# Bug Template

A bug is something broken that needs fixing.

## Sections

### Summary
Brief description of the problem. What is wrong?

### Steps to Reproduce
Exact steps to trigger the bug. Include environment details if relevant.

### Expected Behavior
What should happen when following those steps?

### Actual Behavior
What happens instead? Include error messages, screenshots, or logs if available.

### Severity & Impact
How bad is it? Who is affected and how often?

### Workaround
Any temporary mitigation that users can apply while the fix is pending.
