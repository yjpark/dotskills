# Task Template

A task is a concrete piece of work to complete — a chore, sub-task, or technical action.

## Sections

### Objective
What does this task accomplish? One clear statement of the outcome.

### Context
Why does this need to happen now? Background that explains the motivation.

### Requirements
What specifically needs to be done? List the concrete actions or changes required.

### Acceptance Criteria
How to verify the task is complete. What does "done" look like?

### Dependencies
Other work this depends on, or that this task enables once complete.
